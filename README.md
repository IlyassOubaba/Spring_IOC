# Spring IoC Inversion de Contrôle avec Spring
---

## Description

Ce projet est une application Java simple qui illustre le principe Inversion of Control (IoC) en utilisant le framework Spring.

L'objectif est de comprendre comment Spring gère la création des objets (beans) et l’injection des dépendances entre les différentes couches d’une application Java.

Dans une application classique, les dépendances sont créées manuellement avec l’instruction new.
Avec Spring IoC, c’est le conteneur Spring qui se charge de créer les objets et d’injecter automatiquement leurs dépendances.

---

## Technologies utilisées

**Java 17**

**Spring Framework (Spring Core / IoC)**

**Maven**

**IntelliJ IDEA**
